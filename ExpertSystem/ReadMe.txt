Ce fichier est destiné où utilisateur du système.





==> Notre système est un sytème expert qui fait la recommendation des langages 
de programmations selon des caractéristiques choisies par l'utilisateur, 
en remplissant un formulaire de question.
    Il a été développé en utilisant le langage de programmation Prolog pour la 
rédaction de la base de connaisance et le langage Python pour les autres composantes.

Bibliothèques:
     - Pour l'interface nous avons utilisé la bibliothèque "PyQT5".
     - Nous avons utilisé PySWIP pour relier entre Python et Prolog.


Note: 
     - Les fichiers sources sont disponibles mais pas l'executable.
     - Il faut installer les bibliothèques avant d'executer le code.
